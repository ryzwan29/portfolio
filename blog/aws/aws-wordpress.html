Pengertian Wordpress

WordPress adalah Content Management System (CMS) yang diluncurkan oleh Matt Mullenweg dan Mike Little pada tahun 2004. Sebagai sebuah CMS, WordPress bisa membantu Anda membuat dan mengelola website tanpa coding. Anda bisa menata tampilan, menambahkan fitur, dan membuat konten di website dengan mudah. Jadi, cocok untuk digunakan pemula sekalipun. Tak hanya itu, WordPress juga bisa Anda gunakan secara gratis karena bersifat open-source. Saat ini wordpress sudah di gunakan oleh lebih dari 455 juta website.CyberPanel merupakan web panel yang kita akan pakai pada handout kali ini karena gratis, mudah di gunakan dan memiliki fitur yang lengkap. Salah satunya adalah fitur Auto Installer WordPress yang memungkinkan kita melakukan installasi dengan sekali klik. Bagaimana caranya ? mari ikuti langkah berikut.

"Create Website"

Select Package : pilih package Default
Select Owner : Secara default, owner website adalah admin. Tetapi, kita juga bisa memilih pengguna lain sebagi owner untuk mengelola website.
Test Domain : pilih Disable
Domain Name : Masukan nama domain website untuk install wordpress. Karena kita belum memiliki domain custom yang berbayar, disini kita pakai domain / dns public ec2 saja.
Email : Masukan email yang kita gunakan untuk mengelola website.
Select PHP : Pilih versi PHP yang akan digunakan pada website.
Additional Features : Bagiian ini merupakan fitur tambahan bagi website, kita dapat menambahkan SSL, untuk tambahan kemanan.

List Website

Setelah menyiapkan website untuk install Wordpress, sekarang kita perlu mengakses dan mengkonfiguras website dengan cara masuk ke menu List Website pada sidebar CyberPanel

Main> Website> List Website lalu klik menu "Manage"

Hapus File index.html

Pada halaman Website Manager, scroll kebawah kemudian klik File Manager.

Setelah itu, masuk kedalam direktori public_html dan hapus file index.html dengan cara klik kanan pada file yang akan dihapus kemudian pilih “Delete”

Memilih Wordpress sebagai CMS

Untuk mengakses intaller WordPress,masuk ke menu List Website kemudian klik Manage di bagian kanan atas domain yang kita buat. Pada halaman Manage Website, temukan menu Aplication Installer, lalu pilih WP + LSCache Installer.

Pada halaman ini, Kita perlu mengisi detail instalasi WordPress, yaitu :

  • Blog Title : Judul blog WordPress, Silahkan isi apapun sesuai keinginan kita
  • Login User : Kita perlu mengisi username yang akan digunakan untuk mengelola WordPress.
  • Login Password : Isilah dengan password yang akan digunakan untuk login ke WordPress. Sebaiknya, simpan detail login di catatan pribadi / notepad agar tidak lupa.
  • Email : Masukan email yang akan digunakan untuk login.
  • Path : bagian ini tidak perlu di isi, secara default WordPress akan terintall di home directory.
