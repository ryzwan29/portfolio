CyberPanel

cPanel adalah platform manajemen hosting yang memiliki banyak fitur serta tampilan dasbor yang intuitif. cPanel bertindak sebagai panel kontrol untuk memudahkan pengelolaan layanan hosting, baik itu dari server maupun website. CyberPanel adalah control panel hosting yang didukung oleh OpenLiteSpeed. Web panel ini kaya akan fitur, seperti auto-backup, auto-SSL, FTP server, manajemen PHP, dan lain-lain.

Wordpress

WordPress adalah sistem manajemen konten web. Awalnya dibuat sebagai alat untuk menerbitkan blog tetapi telah berevolusi untuk mendukung penerbitan konten web lainnya, termasuk situs web yang lebih tradisional, milis dan forum Internet, galeri media, situs keanggotaan, sistem manajemen pembelajaran, dan toko online.

NGINX

Nginx adalah webserver dengan lisensi open-source yang memiliki multi fungsi sebagai proxy IMAP/POP3. Web server Nginx terkenal karena stabil, memiliki tingkat performansi tinggi dan minim mengonsumsi sumber daya.

Zabbix

Zabbix agent: Zabbix agent adalah UNIX daemon yang berjalan pada host yang sedang dipantau. Agen ini memberikan informasi, performa pada host dan mengirimkan informasi-informasi tersebut ke Server zabbix.

Host Monitoring

Pada tutorial ini, host yang akan dimonitor adalah Debian12. Untuk OS sendiri, agar dapat dimonitoring menggunakan Zabbix maka harus menggunakan Zabbix Agent. Untuk melakukan monitoring perangkat lain, misalkan router atau perangkat jaringan lainnya maka dapat menggunakan protokol SNMP.

VPC

Virtual Private Cloud atau yang biasa disebut dengan VPC merupakan suatu fitur networking pada Public Cloud. Fitur ini memberikan keamanan dan fleksibilitas yang lebih pada penggunanya. Dengan adanya VPC, pengguna bisa mengatur resource infrastruktur mereka agar lebih cocok dengan kebutuhan setiap penggunanya.

Website Static

Kita akan melakukan intalasi dan konfigurasi web server yang melakukan hosting website dinamis dengan database yang terpisah di subnet private hingga tidak bisa diakses dari internet. Lalu bagaimana cara mengkoneksikan dengan database? Mari kita bahas-!!

Nagios

Nagios merupakan sebuah sistem dan aplikasi monitoring jaringan open – sources yang dibuat oleh Ethan Galstad. Nagios mengawasi host-host dan servis yang telah ditentukan, dan menampilkan peringatan (alert) ketika sesuatu yang buruk terjadi ke Server – seperti server down. Nagios dijalankan dalam sistem operasi Linux.

Nagios-WebServer

Install Nagios-Client di EC2-WebServer agar bisa dimonitoring oleh EC2-Nagios.

Nagios-Database

Install Nagios-Client di EC2-Database agar bisa dimonitoring oleh EC2-Nagios.
