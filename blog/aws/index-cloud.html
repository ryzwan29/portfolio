CyberPanel

cPanel adalah platform manajemen hosting yang memiliki banyak fitur serta tampilan dasbor yang intuitif. cPanel bertindak sebagai panel kontrol untuk memudahkan pengelolaan layanan hosting, baik itu dari server maupun website. CyberPanel adalah control panel hosting yang didukung oleh OpenLiteSpeed. Web panel ini kaya akan fitur, seperti auto-backup, auto-SSL, FTP server, manajemen PHP, dan lain-lain.

Wordpress

WordPress adalah sistem manajemen konten web. Awalnya dibuat sebagai alat untuk menerbitkan blog tetapi telah berevolusi untuk mendukung penerbitan konten web lainnya, termasuk situs web yang lebih tradisional, milis dan forum Internet, galeri media, situs keanggotaan, sistem manajemen pembelajaran, dan toko online.

NGINX

Nginx adalah webserver dengan lisensi open-source yang memiliki multi fungsi sebagai proxy IMAP/POP3. Web server Nginx terkenal karena stabil, memiliki tingkat performansi tinggi dan minim mengonsumsi sumber daya.

Zabbix

Zabbix agent: Zabbix agent adalah UNIX daemon yang berjalan pada host yang sedang dipantau. Agen ini memberikan informasi, performa pada host dan mengirimkan informasi-informasi tersebut ke Server zabbix.
